# h5 Gitar Hero

## x)
- Git tallentaa tilannevedoksia muutoslistojen sijaan.
- Lähes kaikki toiminnot tapahtuvat paikallisesti omalla koneellasi.
- Tiedostot ovat joko muokattuja (modified), seuraavaan tallennukseen poimittuja (staged) tai tietokantaan turvallisesti tallennettuja (committed).

#### git add --all
- Lisää kaikki muutokset versionhallintaan. --all takoittaa muokattuja, poistettuja ja uusia tiedostoja.
#### git commit
- Tallentaa staging-alueella olevat muutokset Gitin historiaa ja luo uuden commitin.
#### git pull
- Hakee muutokset esim. GitHubista ja yhdistää ne omaan projektiin.
#### git push
Lähettää paikalliset commitit etärepositorioon esim. GitHub.
